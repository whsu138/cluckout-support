What if notification permission is denied?

If notification permission is denied, the reminder toggle will be disabled and you'll see a warning message. To fix this:

1. Tap "Open Settings" in the Reminders section
2. Navigate to CluckOut in iOS Settings
3. Enable "Allow Notifications"
4. Return to the app and try enabling reminders again

New in 1.2.2: The app now provides clear guidance and direct access to iOS Settings when permission is denied.

How do I manage my reminders?

1. Go to Settings > Reminders

3. Add new reminders or modify existing ones
4. Enable/disable reminders as needed

Improved in 1.2.2: Better reminder management with automatic permission checking and clearer status indicators. The toggle is now disabled when notification permission is not granted, preventing confusion.

Why is the reminder toggle disabled?

The reminder toggle is disabled when notification permission is not granted. This ensures that reminders only work when properly configured. To enable it:
