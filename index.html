Time Tracking Basics

How do I track my time?

1. Open CluckOut
2. Tap "Clock In" when you start working
3. Tap "Clock Out" when you finish

Enhanced in 1.4.0: Time tracking now includes meaningful animations with logo pulse effects, breathing button animations, and real-time duration updates with estimated pay display. Enjoy haptic feedback on supported devices for a more engaging experience.

How do I edit a time entry?

1. Go to the Summary tab
2. Tap on the entry you want to edit
3. Modify the clock-in/clock-out times with second-level precision
4. Tap Save

Improved: Edit interface now supports more precise time adjustments and provides better validation.

How do I add a manual time entry?

1. Go to the Summary tab
2. Tap the "+" button
3. Enter clock-in and clock-out times with second-level precision
4. Set the hourly rate for this entry
5. Tap Save

Enhanced: Manual entry now supports more precise time input and better rate management.

Enhanced in 1.4.0: Import now includes progress indicators, enhanced error handling, and overnight shift support. Enjoy clean progress overlays and comprehensive validation.

Legacy Format Support: The app automatically detects and supports multiple time formats:

  • Legacy format: HH:mm (e.g., 09:30, 14:45)

What if notification permission is denied?

If notification permission is denied, the reminder toggle will be disabled and you'll see a warning message. To fix this:

1. Tap "Open Settings" in the Reminders section
2. Navigate to CluckOut in iOS Settings
3. Enable "Allow Notifications"
4. Return to the app and try enabling reminders again

New in 1.2.2: The app now provides clear guidance and direct access to iOS Settings when permission is denied.
